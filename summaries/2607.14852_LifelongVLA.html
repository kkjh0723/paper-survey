Lifelong Learning VLA LoRA Gating Replay xArm

제출: 2026년 7월 16일 · 백본 PaliGemma (Gemma 2B VLM + 300M action decoder) · 7-DoF action

한 줄 요약 (TL;DR)

배포된 로봇이 새 조작 스킬을 순차적으로 습득하면서 이전 능력을 잊지 않아야 하는 lifelong VLA 세팅을 다룸. 핵심은 두 가지: (1) Dual-Timescale LoRA Gating — 짧은 학습률의 short-term LoRA와 낮은 학습률의 long-term LoRA를 task-aware gate αt,l(c)로 가중 결합해 plasticity–stability 자동 조율. (2) Cache-Efficient Stochastic Replay — 원 궤적 대신 stop-grad prefix 토큰만 캐시하고 suffix는 매번 새 noise로 재생성. 메모리 167.62→95.70 MiB/task (−43%), 평균 SR 83.2%(ER 70.2, upper bound Joint 89.6), Forgetting Rate 11.4%(ER 19.6, SFT 76.8). 실기 xArm 5-task 순차 학습에서 모든 태스크 80% 이상 유지.

1배경 · 문제의식

범용 VLA는 오프라인 대규모 학습에 강하나 배포 후 새 태스크를 하나씩 습득해야 하는 실제 서비스 로봇 시나리오에서 두 가지 벽에 부딪친다: (a) 이전 태스크의 catastrophic forgetting, (b) 모든 궤적을 저장해 replay하는 방식의 비현실적 메모리 비용. 저자들은 이 plasticity–stability trade-off를 이중 시간 척도(dual-timescale) 학습과 압축 replay로 동시에 공략한다.

2핵심 Contribution

  • Dual-Timescale LoRA Gating — 동일한 base 가중치에 두 LoRA(Δ𝐖sh, Δ𝐖lg)를 붙이되 학습률을 이원화: short-term은 현재 태스크 denoising loss로 빠르게, long-term은 replay·distillation으로 천천히. Frozen prefix 특징 c = Pool(stopgrad(Epre(u))) 기반 gate αt,l(c) = σ(Wg,lφl(c)+bg,l)가 가중치 수준으로 두 LoRA를 결합.
  • Cache-Efficient Stochastic Replay — 궤적 전체가 아닌 stop-grad prefix 토큰과 (state, action) supervision만 저장(reservoir sampling). 매 replay 시 새 diffusion noise (τj', εj')로 suffix를 재계산해 stochastic 다양성 확보. 167.62→95.70 MiB/task (−43%).
  • Masked New-Replay + Distillation 목적함수 — 현재/replay 항을 별도 정규화, teacher snapshot의 detached prediction으로 distillation. ℒt = ℒnew + λℒrep + βℒdist (λ=1.0, β=0.1).

3Method

Dual LoRA + Gating (수식)

각 계층: Δ𝐖̂t,l(c) = (1−αt,l(c))·Δ𝐖sh,t,l + αt,l(c)·Δ𝐖lg,t,l
Short-term (ηsh ≫ ηlg): 현재 태스크 신속 적응 담당. Long-term (ηlg): replay·distillation으로 통합 지식 축적. Gate는 context c에 따라 두 pathway를 태스크 인식적으로 혼합.

Stochastic Replay

스킬-수준 prefix cache ℛt는 reservoir sampling으로 |ℛt| = min(ct, |Ωt|). 캐시 항목당 여러 stochastic suffix 인스턴스 생성: hj,tsuf,rep = Esuf(sj, ỹτj'rep, τj'; Θ̂t(cjrep)). 원 궤적 저장 대비 정보 손실 최소·메모리 절약 동시.

5Ablation (주요 발견)

  • Cache 표현: Raw trajectory replay(167.62 MiB) → latent prefix replay(95.70 MiB). SR 유지·FOR 개선이 동시에 관찰됨. 정보의 압축이 오히려 안정성에 유리.
  • Dual-timescale 이득: Single LoRA(74.6% / 21.78%) → Dual LoRA(83.2% / 11.4%). Short/long 분리 학습 + gate가 plasticity와 stability를 동시에 얻는 핵심.
  • Parameter efficiency: Dual LoRA는 99.97M로 AtomicVLA(226.49M)의 절반 이하. LoRA를 두 개 두는 것이 압도적으로 효율적.
  • Task-aware gate 필요성: Fig. 3에서 LifelongVLA만이 SR/FOR 평면의 상단좌측 이상영역에 근접 — 여타 방법은 plasticity 또는 stability 한쪽에 편중.
