Learning Fine-Grained Bimanual Manipulation with Low-Cost Hardware (ALOHA / ACT)

Tony Z. Zhao, Vikash Kumar, Sergey Levine, Chelsea Finn · Stanford · UC Berkeley · Meta

arXiv abstract PDF 프로젝트 페이지 ACT GitHub arXiv:2304.13705
Bimanual Manipulation Action Chunking Transformer CVAE Imitation Learning

제출: 2023년 4월 23일 · 하드웨어 $20k ALOHA (2× ViperX 6-DoF + 2× WidowX 리더) · Robotics: Science and Systems 2023

한 줄 요약 (TL;DR)

$20k 저가·저정밀 양팔 원격조작 플랫폼(ALOHA)과, k-스텝 action chunk를 CVAE-Transformer로 예측하고 배포 시 시간 축으로 앙상블(temporal ensemble)하는 Action Chunking with Transformers(ACT). 10분(≈50 demos) 시연만으로 케이블 타이 끼우기·건전지 삽입 등 정밀 양팔 6개 실세계 태스크에서 80–90% 성공률을 달성해, "저가 하드웨어 + 학습"이 고정밀 매니퓰레이션에 가능함을 보인 대표작. 이후 등장한 대다수 실로봇 VLA(π0, OpenVLA 등)가 참조하는 ACT baseline의 원류.

1배경 · 문제의식

Fine manipulation(케이블 정렬, 삽입, 뚜껑 열기 등)은 mm 단위 정밀도와 미세한 힘 조율이 필요해 전통적으로 수십만$의 고정밀 로봇·센서가 요구되었다. 저자들은 반대로 묻는다: 저정밀·저가 하드웨어에서 학습만으로 이 격차를 메울 수 있는가? 저가 로봇은 관절 백래시·리피터빌리티가 큰 대신, 정책이 closed-loop로 오류를 관측·보정할 수 있다면 문제 해결이 가능하다는 가설이다.

기존 BC(behavior cloning)는 (1) 복합 오차 누적(compounding error), (2) 시연자의 비-마르코프(non-Markovian) 잠재 의도, (3) 저정밀 하드웨어의 고빈도 지터 때문에 fine task에서 실패한다. ACT는 이 세 문제를 action chunking + temporal ensembling + CVAE latent로 동시에 겨냥한다.

2핵심 Contribution

  • ALOHA 플랫폼 — 총 $20k 미만으로 조립 가능한 저가 양팔 원격조작 시스템(리더–팔로워 4팔, ViperX/WidowX). 도면·CAD·부품 리스트를 오픈해 커뮤니티가 재현 가능.
  • Action Chunking with Transformers (ACT) — 한 번의 예측으로 k=100 스텝의 action chunk를 CVAE로 생성해 오차 누적을 억제하고, 배포 시 여러 시점의 chunk 예측을 지수가중 평균(temporal ensembling)해 저정밀 지터를 완만하게 만든다.
  • 정밀 양팔 6태스크 검증 — 10분(50 demos) 시연만으로 80–90% 성공률. BC-MLP·BeT·RT-1·VINN 등 강력한 최신 imitation baseline들을 큰 폭으로 능가. 이후 fine-manipulation IL의 사실상 표준 백본이 됨.

3Method

ALOHA 하드웨어

리더–팔로워 원격조작 (leader-follower) 구조. 사용자는 2개의 저가 6-DoF WidowX(leader)를 손으로 잡고 움직이면, joint mapping을 통해 2개의 ViperX(follower, 각 7-DoF: 6관절+그리퍼)가 그대로 따라 움직인다. 관측은 4개의 로지텍 웹캠(front, top, 좌·우 wrist)에서 480×640 RGB, 50Hz. 팔 상태(14차원: 좌·우 6관절+그리퍼)도 관측에 포함. 총 부품비 $20k 미만.

ACT: CVAE-Transformer 정책

ACT는 Conditional VAE로 정책을 학습한다. 인코더(BERT-style transformer)는 학습 시에만 사용되며, 관측 없이 k=100 미래 액션 시퀀스를 입력으로 받아 시연자의 스타일 잠재변수 z(가우시안)를 인코딩. 디코더(Transformer encoder-decoder)는 현재 카메라 4장(각 ResNet-18로 인코딩)·관절 상태·z를 입력으로 받아 100-스텝 action chunk를 한 번에 예측한다.

학습 손실: L = L1(â, a) + β·KL(q(z|a)‖N(0,I)). 배포 시 z=0으로 고정(모드 평균), 매 t마다 새 chunk를 예측하지만 각 스텝의 실제 실행 액션은 과거 chunk들의 t번째 예측을 지수가중 평균: â_t = Σ w_i · â_t^(from t-i), w_i ∝ exp(-m·i). 이 temporal ensembling이 저가 하드웨어의 고빈도 지터·outlier 액션을 부드럽게 만든다.

Fine manipulation 시뮬 태스크 8종에서 ACT 평균 44%(scripted-policy 데이터로 30 demos 학습 기준) vs BeT 15%, VINN 6%. 데이터가 매우 적을 때(≤50 demos)의 우위가 특히 큼.

5Ablation (주요 발견)

  • Action chunking (k=100): k=1(비-chunk) 대비 성공률 +30~40pt. k=200으로 더 늘리면 예측 어려워져 소폭 하락. 저자들은 k=100(≈2초)이 sweet spot이라 보고.
  • Temporal ensemble: 없이도 chunking만으로 개선되지만, 지터가 큰 태스크(Ziploc/Cup)에서 +15pt 추가 상승. 배포 시 계산 오버헤드 없음(단순 지수평균).
  • CVAE latent z: 시연자마다 다른 스타일(속도·경로)이 모드 평균되어 실패하는 문제를 완화. z 없이 결정론 회귀만 쓰면 fine task 성공률 절반 이하로 하락.
  • 시연 수 스케일링: 10 demos → 20 → 50으로 늘릴수록 지속 향상, 50 demos 이후 완만한 plateau. "10분 데이터로도 학습 가능"이 핵심 실용 메시지.
