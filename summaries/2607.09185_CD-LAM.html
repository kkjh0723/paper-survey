CD-LAM: Causally Debiased Latent Action Model for Embodied Action Conditioned World Models

Yufan Wei, Kun Zhou, Lingjun Mao 외 · Biwei Huang (교신) · arXiv:2607.09185

arXiv abstract PDF arXiv:2607.09185
World Model Latent Action Causal Debiasing ACWM Robot Adaptation

제출: 2026년 7월 10일 · 백본 2B / 14B ACWM · 96× H100

한 줄 요약 (TL;DR)

Latent Action Model(LAM)이 배경·미접촉 물체 같은 행동과 무관한 시각 요인과 실제 행동 동역학을 얽어버리는 문제(=action-irrelevant bias)를 인과적으로 분리하는 3단계 파인튜닝 프레임워크. SAM3 전경 마스크 기반 embodiment 재구성 + 12-way primitive 대조학습 + zero-transition 캘리브레이션으로 2B·14B 두 스케일 ACWM에서 latent action rollout FDCE를 42% / 26% 개선하고, 로봇 행동 적응을 12× 더 적은 업데이트로 baseline 수준에 도달시킨다. 총 1,000 스텝의 가벼운 파인튜닝만 필요.

1배경 · 문제의식

Action-Conditioned World Model(ACWM)은 embodied 행동에 조건화해 미래 관찰을 시뮬레이션한다. 이때 latent action은 대개 다음 프레임 재구성(reconstruction)만으로 학습되는데, 이 방식은 배경 변화, 카메라 이동, 미접촉 물체의 우연한 상관까지 latent action에 encode해버린다. 결과적으로 (i) 제로 액션 입력에도 embodiment가 계속 움직이는 잔여 모션(residual motion)이 발생하고, (ii) 카메라가 흔들리기만 해도 latent 반응이 크게 튀는 공변량 편향이 나타나며, (iii) 두 궤적 간 target-action 전이 시 action following이 무너진다. 저자들은 이 세 병증을 정량화하는 진단 지표를 새로 정의하고, 이를 최소화하는 인과적 debiasing 목적함수를 제안한다.

2핵심 Contribution

  • Action-irrelevant bias의 정량 진단 — zero-transition response, 카메라 shift 반응(수평·수직), shortcut leakage를 측정하는 새 지표를 정의하고 baseline(DreamDojo)에서 심각한 편향을 실증(표 I).
  • CD-LAM: 3단계 인과 debiasing 파이프라인 — Stage 1(LAM debiasing, 1,000 스텝) → Stage 2(ACWM 재학습, 2,000 스텝) → Stage 3(로봇 액션 → debiased latent 매핑 MLP). SAM3 전경 마스크 기반 embodiment-centric 재구성, 12-way primitive 대조학습, zero-transition 캘리브레이션을 함께 사용.
  • 스케일 & 적응 효율 검증 — 2B / 14B 두 스케일에서 latent action rollout FDCE 42% / 26% 개선, PSNR 동시 상승. 로봇 액션 적응 단계에서 3k–6k 스텝만으로 baseline 50k 스텝 수준에 도달(12× 효율).

3Method

3단계 파이프라인

Stage 1 — LAM Debiasing: 사전학습된 DreamDojo LAM을 세 가지 목적으로 파인튜닝. (a) embodiment-centric reconstruction: SAM3로 팔·손·조작물 전경 마스크를 얻고 픽셀 MSE에 αfg ≫ αbg 가중. (b) action-centric contrastive: 궤적을 12개 coarse primitive(예: 좌/우/상/하/파지 등)로 라벨링해 softplus loss로 같은 primitive → 끌어당김, 다른 primitive → 밀어냄. (c) latent space calibration: free-bit KL + zero-transition anchoring (프레임이 그대로인 클립의 latent 노름이 일반 전이의 mzero·RMS 이하가 되도록 제약).

Stage 2 — ACWM 재학습

debiased LAM이 만든 latent action으로 2B / 14B ACWM을 2,000 스텝 파인튜닝. 조건화 형식은 원본과 동일하게 유지되어 downstream 인터페이스가 바뀌지 않는다.

Stage 3 — Robot Action Adaptation

실제 로봇 액션(관절/그리퍼)을 debiased latent space에 매핑하는 가벼운 MLP bridge만 학습. 이 단계에서 baseline은 50k 스텝 필요했던 반면 CD-LAM은 3k–6k 스텝이면 동일 수준을 넘어선다.

5Ablation (표 IV·V)

  • Debiasing 데이터 스케일 (2B, 표 IV): baseline 12.63 → 1h 8.91 → 100h 8.24 → 1000h 7.97 FDCE. 1시간 규모만으로도 큰 이득, 100h 이후 saturate.
  • Embodiment 가중 제거: FDCE +1.17 px 악화. 전경 강조가 없으면 배경 재구성에 힘이 쏠려 다시 편향 유입.
  • Contrastive 제거: FDCE +1.84 px 악화 — 세 목적함수 중 가장 큰 기여.
  • Zero-transition 캘리브레이션 제거: 카메라 shift 반응이 4.8–6.3× 폭증. "가만히 있음"을 명시적으로 anchor하는 항이 공변량 편향 억제의 핵심.
