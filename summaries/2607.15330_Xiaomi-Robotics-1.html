Xiaomi-Robotics-1: Scaling Vision-Language-Action Models with over 100K Hours of Real-World Trajectories

Jun Guo, Piaopiao Jin, Jason Li 외 30여 명 · Xiaomi Robotics Team

arXiv abstract PDF 프로젝트 페이지 arXiv:2607.15330
VLA Foundation Model UMI Scaling Law Flow Matching

제출: 2026년 7월 16일 · 백본 Qwen3-VL + DiT (MoT) · 2B/5B/10B 세 규모

한 줄 요약 (TL;DR)

UMI 그리퍼로 수집한 10만 시간+ 실세계 조작 궤적과 자동 라벨링 파이프라인으로 학습한 VLA 파운데이션. VLM(Qwen3-VL) + Flow-matching DiT를 Mixture-of-Transformers로 결합하고, "pre-training → post-training" 2단계 레시피와 데이터·모델 스케일링 커브를 실증. RoboCasa365 평균 57.4 (SOTA 46.6), RoboDojo 20.07 (SOTA 13.07), 저데이터 파인튜닝에서 π0.5(40%) 대비 75% 성공률.

1배경 · 문제의식

VLA는 언어·시각을 로봇 action으로 매핑하지만, 대규모 학습에 필요한 실세계 조작 궤적은 여전히 병목이다. 원격조작(teleop)은 비용이 크고, 시뮬레이션은 sim-to-real 갭이 있으며, 인간 영상(SSv2, Ego4D)은 low-level action 라벨이 없다. Xiaomi 팀은 UMI(Universal Manipulation Interface)라는 그리퍼 형태 휴대장치로 인간이 직접 조작하며 action-쌍이 있는 실세계 조작 궤적을 확장 가능하게 수집하고, 이를 10만 시간 이상으로 스케일하여 GPT/LLaMA 스케일 법칙에 준하는 로봇 파운데이션 모델의 스케일링 실증을 목표로 한다.

2핵심 Contribution

  • 100K시간+ UMI 사전학습 코퍼스 — Qwen3.5-27B 기반 자동 라벨링 파이프라인(producer-consumer)으로 상태 전이를 자연어로 캡셔닝, 약 2주 만에 대규모 라벨링 완료.
  • MoT 기반 VLA 아키텍처 — Qwen3-VL VLM + Flow-matching DiT(action head)를 Mixture-of-Transformers로 결합. Choice policy로 VLM에 보조 action supervision, action 토큰은 DiT attention에서 제외해 shortcut 차단.
  • Pre-training + Post-training 2단계 학습 — UMI로 broad capability 학습 후, 10K시간 크로스-임보디먼트(모바일 조작기·dual-arm·오픈소스 데이터)로 embodiment alignment.
  • 스케일링 커브 실증 — 데이터 0%→100%: 26%→75%, 모델 2B→10B: 61%→75%→79%. 사전학습 데이터·모델 크기 모두에서 일관된 성능 향상.
  • 다중 벤치마크 SOTA — RoboCasa365 평균 57.4(+10.8pp), RoboDojo 평균 20.07(+7.0pp), VLABench 75.6% ID 성공률. 저데이터 파인튜닝에서 π0.5 대비 큰 우위.

3Method

아키텍처: Mixture-of-Transformers (MoT)

VLM(Qwen3-VL)이 관측·언어 지시를 인코딩하고, DiT(Diffusion Transformer)가 flow-matching으로 action chunk를 생성. DiT는 로봇 proprioceptive state와 VLM의 KV cache를 조건으로 받으며, action 관련 토큰은 DiT의 attention에서 제외되어 예측 shortcut을 방지한다. Choice policy가 VLM 내부에서 보조 action supervision을 제공해 VLM 표현도 action-aware하게 만든다.

학습 목적

Flow-matching loss(action 궤적 예측, beta-분포 timestep 샘플링) + Regression loss(winner-takes-all: 최우수 후보 action만 supervision) + NTP loss(vision-language 공동학습, 비율 1:9). 전체 목적: L = LFlow + LRegression + λ·LNTP, λ=0.1.

Auto-labeling 파이프라인

Qwen3.5-27B가 UMI 궤적을 시청하며 상태 전이(state transition)를 자연어로 캡셔닝. Producer-consumer 아키텍처로 GPU/CPU 활용을 극대화, 10만 시간 규모 데이터를 약 2주 만에 라벨링 완료. VLM 학습 시 (관측, 언어 지시) 쌍을 대량 확보하는 것이 핵심.

5Ablation (주요 발견)

  • 사전학습 데이터 스케일링: 데이터 0→100%에서 실세계 성공률 26→75%로 거의 3배. 스케일링이 계속 이득을 준다.
  • 모델 크기 스케일링: 2B→5B에서 큰 점프(61→75), 5B→10B는 완만한 상승(75→79). Chinchilla 유사한 데이터-모델 균형 존재 시사.
  • Auto-labeling의 가치: 언어 라벨 없는 UMI 궤적만으로는 VLM alignment가 안 됨. 자동 캡셔닝이 (관측, 언어) 페어를 만들어 VLM 활용을 가능케 함.
  • Cross-embodiment post-training: UMI로만 학습한 모델은 실 로봇에 못 씀. post-training 단계의 embodiment alignment가 필수.
