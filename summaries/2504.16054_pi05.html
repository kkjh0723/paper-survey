π0.5: a Vision-Language-Action Model with Open-World Generalization

Physical Intelligence (Kevin Black, Noah Brown, Danny Driess, Chelsea Finn, Sergey Levine, Karol Hausman, Karl Pertsch 외)

arXiv abstract PDF 프로젝트 블로그 arXiv:2504.16054
VLA Open-World Generalization Co-Training Hierarchical Inference Mobile Manipulation

제출: 2025년 4월 22일 (v1) · Physical Intelligence · 이동 조작 로봇(듀얼 암 + 홀로노믹 베이스)

한 줄 요약 (TL;DR)

π0의 후속으로, 완전히 새로운 실제 가정(학습 데이터에 없는 집)에서 부엌·침실 청소 같은 10~15분 장기 조작을 end-to-end로 수행하는 첫 VLA. 핵심은 무차별 로봇 데이터 수집이 아니라 이질적 데이터 공동학습(co-training) — 사전학습 데이터의 97.6%가 이동조작이 아닌 소스(타 로봇, 웹 데이터, 실험실 데이터)다. 계층적 추론(고수준 서브태스크 예측 + 저수준 행동 생성)을 단일 모델로 통합하고, 이산 토큰(FAST) 사전학습 + flow matching 연속 추론의 하이브리드 행동 표현을 쓴다. 학습 환경 수를 104개까지 늘리면 테스트 가정에서 직접 학습한 오라클(≈92%)에 근접(≈90%)해, 데이터 다양성이 일반화를 이끈다는 것을 보인다.

1배경 · 문제의식

로봇이 실제로 쓸모 있으려면 실험실이 아닌 현실 세계에서 유의미한 작업을 해야 한다. π0 같은 기존 VLA는 학습 데이터와 유사한 환경에서는 잘 작동하지만, 전혀 새로운 집(새 배치·새 물체·새 구조)으로의 진정한 open-world 일반화, 특히 "부엌을 청소하라" 같은 장기(long-horizon) 과제에서는 실패한다. 이전 연구가 보여준 일반화는 물건 집기·서랍 열기 같은 단순·단기 스킬에 한정됐고, 장기 과제에 필요한 폭넓은 상식/의미 이해는 로봇 데이터만의 무차별 수집으로는 달성 불가능하다.

2핵심 Contribution

  • 새 가정에서의 장기 조작 최초 실증 — 해당 환경 학습 데이터 없이도 완전히 새로운 집에서 10~15분 장기·정교 조작을 수행하는 end-to-end 학습 로봇 시스템을 처음 시연.
  • 이질적 데이터 공동학습(co-training) 프레임워크 — 사전학습 데이터의 97.6%가 이동조작 이외 소스(타 로봇, 웹 데이터, 실험실 데이터)로, 크로스-임바디먼트·웹 지식을 결합.
  • 계층적 통합 아키텍처 — 하나의 모델이 고수준 의미적 서브태스크 예측과 저수준 행동 추론을 동시에 수행.
  • 하이브리드 행동 표현 — 효율적 사전학습을 위한 이산 토큰화(FAST)와 배포 시 연속 행동 추론을 위한 flow matching을 결합.
  • 일반화 스케일링 검증 — 일반화 성능이 학습 환경 다양성에 따라 단조 증가하며, 공동학습 구성요소 각각이 필수임을 ablation으로 입증.

3Method

공동학습 데이터 소스

사전학습은 5종의 이질적 데이터를 함께 사용한다. 이동조작(MM) 데이터는 실제 가정에서 수집한 ~400시간에 불과하고, 나머지 97.6%는 타 로봇·실험실·웹에서 온다.

데이터출처역할
MM (이동조작)실제 가정 ~400시간목표 도메인
ME (다환경)다양한 가정의 비이동 로봇크로스-임바디먼트 전이
CE (실험실 크로스-임바디먼트)테이블탑 태스크, OXE다양한 조작 프리미티브
HL (고수준)수동 서브태스크 주석의미적 추론
WD (웹 데이터)CapsFusion, COCO, VQAv2, Cambrian-7M시각·의미 grounding

후처리(post-training)에서는 전문가 사용자의 언어 지도인 VI(구두 지시)가 추가되며, 고수준 데이터의 약 11%로 소량이지만 성능에 결정적이다.

계층적 추론 구조

예측을 다음과 같이 분해한다: π(actions, subtask | obs, task) = π(actions | obs, subtask) × π(subtask | obs, task). 고수준 추론이 의미적 서브태스크(예: "접시를 집어라")를 출력하고, 저수준 추론이 그 서브태스크를 조건으로 연속 행동 청크를 생성한다. 단일 트랜스포머가 둘 다 처리하되, 연속 예측용 "action expert" 가중치를 별도로 둔다.

하이브리드 행동 표현

사전학습: FAST 토크나이저(이산 토큰) 사용 — 확장성 좋은 next-token 예측이 장점이나 추론 시 자기회귀 디코딩이 비쌈. 후처리: flow matching을 추가해 실시간 연속 행동 생성(10 denoising step) + 텍스트 생성 능력 유지. 후처리 손실은 L = H(text) + α·‖noise_target − action_pred‖² (α=10.0).

4실험 결과

새 가정에서의 일반화

학습에 없던 완전히 새로운 집 3곳(부엌 3 + 침실 3, 총 6태스크, 태스크당 10회)에서 평가. "싱크에 그릇 넣기" 약 70~85% 진행률, "서랍에 물건 넣기" 약 80~90%, "빨래 바구니" 약 75~85% 완료. 에피소드당 2~5분 다단계 과제로, 논문은 이를 "기존 VLA가 보인 결과를 크게 상회"한다고 밝힌다.

환경 다양성에 따른 스케일링

학습 환경 수를 3 → 12 → 22 → 53 → 82 → 104개로 늘리며 평가. 성능이 단조 증가하며, 104개 환경 모델은 테스트 집에서 직접 학습한 오라클과 사실상 동등한 수준에 도달한다.

설정평균 성능(근사)
테스트 가정 직접 학습 (오라클)≈ 92%
104개 환경 공동학습 (test 데이터 0)≈ 90%

언어 지시 따르기(Fig. 9): 104개 환경에서 학습 카테고리 물체 ≈85%, 미학습(OOD) 물체 ≈65%. 둘 다 환경 수 증가에 따라 꾸준히 향상.

모델 비교

π0.5는 원본 π0 및 공동학습 없는 강화판 π0-FAST+Flow를 4개 벤치 태스크 전반에서 뚜렷하게 능가한다(Fig. 12).

5Ablation (주요 발견)

  • 웹 데이터(WD) 제거: 모의 가정 태스크엔 영향 미미하나 OOD 물체 일반화에서 크게 저하 — 웹 데이터가 폭넓은 물체 의미 지식을 제공.
  • 다환경(ME) 데이터 제거: 태스크 진행률 약 15~25%p 급락 — 다른 가정으로부터의 크로스-임바디먼트 전이가 핵심.
  • 실험실 CE 데이터 제거: 유의미한 저하 — 다양한 조작 프리미티브가 필수.
  • ME·CE 모두 제거: 사실상 붕괴 — 이동조작 데이터만으로는 불충분, 이질적 소스의 필요성 입증.
  • 고수준 추론·VI: 명시적 고수준 추론이 최고, 구두지시(VI) 제거 시 큰 폭 저하. 흥미롭게도 π0.5는 사람 고수준 주석자보다 더 잘 수행 — 학습된 계층적 분해가 효과적 추상화를 포착함을 시사.
