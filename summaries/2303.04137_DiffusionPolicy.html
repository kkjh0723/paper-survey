Diffusion Policy: Visuomotor Policy Learning via Action Diffusion

Cheng Chi, Zhenjia Xu, Siyuan Feng, Eric Cousineau, Yilun Du, Benjamin Burchfiel, Russ Tedrake, Shuran Song · Columbia University · Toyota Research Institute · MIT · Stanford

arXiv abstract PDF 프로젝트 페이지 GitHub arXiv:2303.04137
Diffusion Policy Visuomotor Learning DDPM Imitation Learning Manipulation

제출: 2023년 3월 7일 (v1) · 최신본 2024년 3월 14일 (v5) · RSS 2023 · DDIM 10-step 추론(0.1s, RTX 3080)

한 줄 요약 (TL;DR)

로봇 정책을 명시적 함수가 아니라 행동에 대한 조건부 denoising diffusion 과정으로 표현하는 visuomotor 정책 학습법. 관찰을 조건으로 하는 action diffusion과 receding horizon control·시계열 diffusion transformer를 결합해 멀티모달 행동 분포, 고차원 action 시퀀스, 학습 안정성을 동시에 해결. 4개 벤치마크의 15개 태스크에서 기존 SOTA 대비 평균 성공률 46.9% 향상.

1배경 · 문제의식

행동 복제(behavior cloning)로 visuomotor 정책을 학습할 때 로봇 행동 예측은 세 가지 근본적 난제를 갖는다: (1) 사람 시연은 같은 상황에서도 여러 경로가 가능한 멀티모달 분포를 가지는데, 단봉(unimodal) 회귀나 명시적 정책은 이를 평균화해 무너뜨린다; (2) 로봇 행동은 관절·엔드이펙터·시퀀스가 얽힌 고차원 연속 action space를 가진다; (3) 에너지 기반 정책(EBM) 등 암시적 정책은 표현력은 있으나 학습이 불안정하다. 저자들은 이미지 생성에서 성공한 denoising diffusion을 행동 예측에 적용해 이 세 문제를 한 번에 다룬다.

2핵심 Contribution

  • Action Diffusion으로 정책 표현 — 정책을 행동에 대한 조건부 denoising diffusion 과정으로 정의해, 스코어 함수의 gradient 만으로 멀티모달 분포와 고차원 action을 안정적으로 표현.
  • Receding Horizon Control과의 결합 — 고차원 action 시퀀스를 한 번에 예측하되 일부만 실행하고 재계획하는 receding-horizon 방식으로 장기 일관성과 실시간 반응성을 동시에 확보.
  • Visual Conditioning — 관찰을 결합 분포의 일부가 아니라 조건으로 취급해, 매 denoising step마다 관찰 특징을 재추론할 필요를 없애 실시간 추론 비용을 크게 절감.
  • 시계열 Diffusion Transformer — over-smoothing을 완화하는 트랜스포머 기반 diffusion 구조로 고주파 행동 변화·속도 제어가 필요한 태스크에 대응.

3Method

DDPM 기반 Action 생성

정책은 관찰 Ot를 조건으로 하는 조건부 DDPM으로, 순수 노이즈에서 시작해 K번의 denoising을 거쳐 action 시퀀스를 생성한다. 학습은 노이즈 예측 손실(예측 노이즈와 실제 노이즈의 MSE)을 최소화한다. 추론은 DDIM을 써서 학습 100 iteration / 추론 10 iteration으로 가속, RTX 3080에서 0.1s 지연으로 실시간 동작.

Receding Horizon Control (관찰/예측/실행 horizon)

매 스텝에서 관찰 horizon To 만큼의 관찰을 조건으로 예측 horizon Tp 길이의 action 시퀀스를 예측한 뒤, 그중 실행 horizon Ta 만큼만 실행하고 다시 관찰·재예측한다. 이렇게 하면 장기 계획성과 폐루프 반응성 사이의 균형을 맞춘다.

네트워크 구조 (CNN vs Transformer)

CNN 기반
1D 시계열 U-Net + FiLM 조건화. 하이퍼파라미터에 둔감해 기본값으로 잘 동작
Transformer 기반
action 임베딩에 causal attention. 고주파·속도 제어 태스크에서 over-smoothing 완화
Visual Encoder
수정된 ResNet-18 + spatial softmax pooling, BatchNorm 대신 GroupNorm으로 학습 안정화
핵심 설계 포인트: 관찰을 조건으로만 넣으므로 시각 특징을 매 denoising 반복마다 다시 계산할 필요가 없어, 고차원 action 시퀀스를 예측하면서도 실시간 추론이 가능하다.

5Ablation (주요 발견)

  • 멀티모달 표현: diffusion 정책은 같은 상황에서 여러 유효 경로(mode)를 무너뜨리지 않고 각각을 표본화 — 회귀·단봉 정책이 평균화로 실패하는 지점에서 우위.
  • Receding horizon: 예측 horizon Tp가 너무 짧으면 근시안적, 너무 길고 실행 horizon Ta가 크면 반응성이 떨어짐 — 적절한 예측/실행 horizon 조합이 성능에 중요.
  • 조건화 vs 결합 모델링: 관찰을 조건으로 다루면 정확도를 유지하면서 추론 비용을 낮춤(실시간 0.1s).
  • 구조 선택: CNN 기반은 하이퍼파라미터에 강건해 기본값으로 잘 동작, Transformer 기반은 고주파·속도 제어 태스크에서 over-smoothing 완화에 유리.

세부 ablation 수치는 원문 표를 참조. 위 항목은 논문의 정성적 발견을 정리한 것이다.
