RT-2: Vision-Language-Action Models Transfer Web Knowledge to Robotic Control

Anthony Brohan, Noah Brown 외 (Google DeepMind Robotics 팀, 55명) · Karol Hausman · Sergey Levine · Chelsea Finn · Brian Ichter

arXiv abstract PDF 프로젝트 페이지 arXiv:2307.15818
VLA (원조) PaLI-X 55B PaLM-E 12B Action-as-Token Emergent Skills

제출: 2023년 7월 28일 (v1) · Google DeepMind · 실기 6,000회 평가

한 줄 요약 (TL;DR)

인터넷 규모 VLM(PaLI-X 55B, PaLM-E 12B)에 로봇 액션을 텍스트 토큰으로 표현해 VQA + 로봇 데이터를 co-fine-tuning한 최초의 대규모 VLA. 미지 물체·배경·환경 일반화가 RT-1 대비 ~2×, 상징 이해·추론·인간 인식 등 emergent 능력에서 ~3× 향상. Language Table 시뮬 90%(이전 SOTA 77%). 6,000회 실기 평가로 검증. Chain-of-thought로 "즉석 망치로 뭘 쓸까?"·"목마른 사람에게 뭘 줄까?" 같은 다단계 추론을 자연스럽게 수행.

1배경 · 문제의식

기존 로봇 정책은 로봇 데이터 안에서만 학습돼 웹 규모의 상식·상징·추론 지식이 없다. RT-1처럼 임팩트 있는 실기 정책도 학습 셋을 넘는 대상·상황엔 취약. 저자들은 이미 웹 이미지+텍스트로 사전학습된 대형 VLM(PaLI-X 55B, PaLM-E 12B)을 로봇 데이터에 공동 파인튜닝하면 이 웹 지식이 제로샷 시맨틱 일반화·추론·상징 이해로 전이될 수 있다고 가설한다.

2핵심 Contribution

  • Action-as-Text 표현 — 로봇 액션(6-DoF 델타 pose + 그리퍼 + 종료 플래그)을 각 차원을 256 bin으로 discretize해 VLM 텍스트 어휘의 토큰으로 매핑. VLM의 언어 디코딩 파이프라인을 그대로 재사용해 로봇 액션을 생성.
  • VQA·로봇 데이터 Co-fine-tuning — 파인튜닝 시 로봇 궤적만이 아니라 원본 VQA/캡셔닝 데이터도 계속 함께 학습. 이 방식이 파국적 망각을 막고 웹 지식을 유지해 미지 개념 일반화가 크게 향상.
  • Emergent capabilities 실증 — 상징 이해(숫자·아이콘 위 배치), 추론(가장 작은/큰/가까운 물체 선택), 인간 인식(예: Taylor Swift 사진)에서 RT-1 대비 ~3×. Chain-of-Thought로 다단계 시맨틱 추론(도구 대용·음료 추천 등) 성공.

3Method

Action Tokenization

로봇 액션 벡터를 terminate ∆x ∆y ∆z ∆roll ∆pitch ∆yaw grip 8개 차원으로 정의하고, 각 차원을 256 bin으로 균등 이산화. PaLI-X는 이미 자연스레 정수 토큰이 있어 그대로 매핑, PaLM-E는 자주 안 쓰는 어휘 256개를 재할당. 최종 출력 시퀀스: "1 128 91 241 5 101 127 217"처럼 단일 토큰 시퀀스.

Chain-of-Thought (PaLM-E 변형)

PaLM-E 12B 변형은 액션 토큰 직전에 "Plan: <자연어 계획>. Action: <토큰>" 형태로 학습해, 실행 전 중간 자연어 추론을 생성. "즉석 망치로 뭘 쓸까?" → "돌"이라고 계획한 뒤 그 물체를 집는 식.

Language Table (시뮬)

기존 SOTA(BC-Z 등) 77% → RT-2 90%. 실기 미지 물체(케첩·바나나 등)에도 zero-shot 성공.

총 실기 평가

6,000회 트라이얼 규모의 실기 평가로 위 결과들을 통계적으로 검증.

5Ablation (주요 발견)

  • Co-fine-tuning의 필요성: 로봇 데이터 only 파인튜닝은 unseen 일반화가 크게 저하 — VQA/이미지-텍스트 데이터를 함께 유지해야 웹 지식이 유지된다.
  • 모델 크기 스케일: PaLI-X 5B < 55B, PaLM-E도 크기↑로 emergent가 강해짐 — VLM 규모가 로봇 일반화의 주요 요인.
  • Chain-of-Thought: PaLM-E 12B에서 CoT를 학습시키면 다단계 추론(도구 대용, 음료 추천 등)이 안정적으로 개선.
  • 속도-크기 트레이드오프: 55B는 표현력 최고지만 실행 3Hz대로 낮음, 5B는 10Hz 이상 — 실기 응답성엔 소형이 유리.
