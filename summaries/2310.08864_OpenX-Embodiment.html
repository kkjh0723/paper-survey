한 줄 요약 (TL;DR)

21개 기관이 모은 22개 로봇 embodiment · 60개 데이터셋 · 100만+ 궤적(527 스킬, 160,266 태스크)을 표준 포맷으로 통합한 Open X-Embodiment 데이터셋과, 이를 공동학습한 범용 정책 RT-X를 공개. RT-1-X는 소규모 데이터셋에서 개별 로봇 전용 모델 대비 성공률 평균 50% 향상(positive transfer), RT-2-X(55B)는 창발 스킬 평가에서 RT-2를 27.3% → 75.8%(약 3배)로 끌어올려 이종 로봇 데이터가 능력을 확장함을 입증.

1배경 · 문제의식

대규모 언어·비전 모델은 다양한 데이터로 학습한 하나의 큰 모델이 개별 특화 모델을 능가함을 보였다. 그러나 로봇 학습은 여전히 로봇·태스크·환경마다 별도 모델을 처음부터 학습하는 것이 관행이다. 저자들은 "여러 로봇 플랫폼의 데이터로 학습한 단일 정책이, 각 로봇 전용 데이터로만 학습한 정책보다 더 잘 동작할 수 있는가?"라는 positive transfer(양의 전이) 가설을 검증한다. 이를 위해서는 (1) 여러 기관·로봇의 데이터를 표준 포맷으로 모으는 것과, (2) 이종 embodiment를 흡수할 수 있는 고용량 모델이 모두 필요하다.

2핵심 Contribution

  • Open X-Embodiment 데이터셋 공개 — 21개 기관 협력으로 22개 로봇 embodiment · 60개 데이터셋 · 100만+ 궤적(527 스킬, 160,266 태스크)을 통합. 모두 표준화된 RLDS/TFDS 포맷으로 배포해 재현성 확보.
  • RT-X 범용 정책 학습 — 기존 RT-1·RT-2 아키텍처를 X-embodiment 데이터로 공동학습해 RT-1-X·RT-2-X를 구축. 이종 로봇 경험이 positive transfer를 만들어냄을 실증.
  • 모델 용량이 전이의 열쇠임을 규명 — 작은 RT-1-X는 대규모 데이터에서 과소적합하지만, 고용량 RT-2-X(55B)는 다양한 embodiment 데이터를 흡수해 창발적 스킬과 강한 in-domain 성능을 동시에 달성.

3Method

데이터셋 구성 (Open X-Embodiment)

21개 기관이 보유한 기존 로봇 데이터셋을 하나의 컬렉션으로 통합. 각 데이터셋은 서로 다른 로봇 형태(단일팔·양팔·모바일 등), 액션 공간, 카메라 시점을 갖지만, 공통 표준 포맷으로 변환해 학습 파이프라인에서 균일하게 다룰 수 있게 함.

배포 포맷
표준화 데이터 포맷 (RLDS 기반)

RT-1-X (효율적 트랜스포머 정책)

RT-1(3,500만 파라미터급 트랜스포머)을 그대로 사용하되, 입력으로 X-embodiment 데이터를 공동학습. 소규모 로봇 데이터셋들에서 양의 전이 효과를 검증하는 데 초점.

RT-2-X (VLA, web-scale 공동학습)

RT-2는 웹 규모 vision-language 데이터로 사전학습된 대형 VLM을 action을 토큰으로 출력하도록 파인튜닝한 VLA. RT-2-X는 이 VLM(최대 55B)을 로봇 데이터 + 웹 데이터로 공동 파인튜닝(co-fine-tuning)하며, Open X-Embodiment 데이터를 함께 학습해 여러 로봇을 아우르는 범용 정책을 만든다.

핵심 통찰: 동일한 이종 데이터라도 모델 용량이 부족하면(RT-1-X) 대규모 데이터에서 과소적합해 전용 모델보다 나빠질 수 있으나, 고용량 RT-2-X는 다양성을 흡수해 강한 성능과 창발 능력을 동시에 얻는다. 웹 사전학습과 history conditioning도 전이의 필수 요소로 지목된다.

  • 모델 크기: 55B(75.8%) ≫ 5B(44.4%) — 클수록 강한 전이·창발.
  • History conditioning: 2프레임 사용 44.4% vs 미사용 14.5% — 시계열 조건화가 큰 이득.
  • 웹 사전학습: 있으면 48.7%대, 없으면 0% — VLA 창발 능력에 필수.
  • 공동 파인튜닝: 웹 데이터와 함께 co-fine-tuning하는 것이 로봇 데이터만 파인튜닝하는 것보다 유리.
