VLA-JEPA: Enhancing Vision-Language-Action Model with Latent World Model

Jingwen Sun, Wenyao Zhang (공동 1저자) 외 · USTC · SJTU · Tsinghua · Zhongguancun Academy · EIT Ningbo · 교신저자 Xin Jin, Zhibo Chen

arXiv abstract PDF 프로젝트 페이지 GitHub HuggingFace arXiv:2602.10098
VLA World Model JEPA Latent Action Robustness

제출: 2026년 2월 17일 (v2) · 백본 Qwen3-VL · 8× A100

한 줄 요약 (TL;DR)

영상 기반 latent-action 사전학습의 4가지 함정(픽셀 목표의 외형 편향, 실세계 영상의 잡음 모션 증폭, 정보 누출(information leakage), 복잡한 다단계 파이프라인)을 진단하고, 이를 JEPA(Joint-Embedding Predictive Architecture) 방식의 누출 없는(leakage-free) 상태 예측으로 해결한 VLA 사전학습 기법. 픽셀 복원 없이 미래 latent state를 예측·정렬해 행동 관련 전이 의미(transition semantics)를 학습하며, 단일 단계 파이프라인으로 단순화. LIBERO 평균 97.2, 강건성 벤치 LIBERO-Plus 79.5(차순위 OpenVLA-OFT 69.6)로 큰 폭 우위.

1배경 · 문제의식

영상으로부터 latent action을 사전학습해 VLA에 주입하는 접근이 최근 유행이나, 저자들은 네 가지 실패 모드를 지적한다: (1) 픽셀 수준 목표는 표현을 행동보다 외형(appearance)에 치우치게 한다; (2) 실세계 영상은 카메라 이동·배경 등 잡음/무관 모션(nuisance motion)을 증폭한다; (3) 미래 관찰을 학습기에 입력으로 넣으면 "latent action"이 미래 자체를 인코딩하는 지름길(shortcut)로 붕괴하는 정보 누출이 발생한다; (4) 다단계 학습 파이프라인은 복잡하고 취약하다.

2핵심 Contribution

  • Latent-action 사전학습의 함정 분석 — 픽셀 목표 편향, 잡음 모션 증폭, 정보 누출, 파이프라인 취약성의 4대 실패 모드를 규명.
  • VLA-JEPA: 누출 없는 상태 수준 JEPA 사전학습 — 픽셀 복원·정보 누출 없이 미래 latent state를 예측·정렬. target encoder가 미래 프레임으로 target을 만들되, student 경로는 현재 관찰만 보고 미래 정보는 오직 지도 신호(target)로만 사용.
  • 단순한 워크플로 + 강건성 향상 — JEPA 사전학습 → action-head 파인튜닝의 2단계 레시피로 LIBERO, LIBERO-Plus, SimplerEnv, 실로봇 전반에서 일관된 강건성·일반화 향상.

3Method

백본 & 토큰 설계

핵심 VLM으로 Qwen3-VL(SigLIP-2 비전 인코더) 사용. 두 종류의 학습형 특수 토큰 도입: ⟨latenti⟩(시점 i의 상태 전이 si→si+1 인코딩)와 ⟨action⟩. 각 ⟨latenti⟩는 K번 복제되어 가변 길이 latent action 인코딩을 가능하게 함.

World State Encoder

동결된 자기지도 V-JEPA2 인코더 F(·)로 각 단일 시점 프레임을 인코딩하고, 동일 에피소드의 여러 시점을 벡터 concat으로 통합해 world-state 표현 sti = ‖v F(Iv,ti)을 구성.

누출 없는 Latent Action 사전학습 (World Modeling)

VLM은 초기 시점 t0의 멀티뷰 관찰 + 언어 지시 ℓ를 받아 ⟨latenti⟩ 토큰을 world dynamics를 요약하는 zti로 매핑. 자기회귀 트랜스포머 world model이 다음 상태 청크를 예측: ŝ = pWMθ(st0:i, zt0:i). time-causal attention 사용 — 한 시점 내부에서는 양방향, 시점 간에는 엄격한 causal(미래 마스킹).

목적함수는 의미 공간의 예측 log-likelihood ELBO 최대화. 동결 V-JEPA2가 결정론적 임베딩을 주므로 KL 항이 사라지고 ELBO는 latent-space 복원 손실로 귀결: LWM = Σ(ŝtk − stk). 핵심: 미래 프레임은 VLM의 입력이 절대 아니며, 오직 V-JEPA2 target 생성에만 쓰여 누출 shortcut을 제거한다.

Action 예측 (Flow Matching, 로봇 데이터)

⟨action⟩ 토큰을 latent-action 토큰 뒤에 붙이고 causal attention으로 전역 action-conditioning 표현 za를 생성. 조건부 flow-matching action head가 연속 action 궤적을 모델링: at=(1−t)ε+t·a0:H, 벡터장 vθ를 LFM=E‖vθ−(a−ε)‖²로 학습. 로봇 데이터 전체 목적: L = LFM + β·LWM. 인간 영상(action-free)은 정렬 손실만, 로봇 데이터는 정렬+행동 예측 손실로 교차 도메인 공동 사전학습 지원.

5Ablation (주요 발견)

  • 인간 영상의 역할: LIBERO/SimplerEnv엔 영향 미미(오히려 SimplerEnv는 도움), 그러나 LIBERO-Plus 강건성엔 큰 이득(79.5 vs 62.9). 인간 영상 비율↑일수록 강건성 일관 향상. → 새 동작을 가르치기보다 기존 스킬의 강건성/안정성을 높임.
  • 통합 사전학습: 단일 단계 통합이 기존 2단계 패러다임 상회. Attention 시각화(그림 6)에서 VLA-JEPA는 팔·손·조작 물체에 정확히 주의, LAPA는 과도하게 조밀한 무관 디테일(누출), UniVLA는 배경 의미에 과주의.
  • 미래 영상 horizon (표 4): T=8이 최적(평균 96.1), T=4는 94.8, T=16은 95.5. 영상 horizon이 action horizon에 가까울 때 성능 최고.
