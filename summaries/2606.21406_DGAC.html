Robot Self-Improvement via Human-Video Dynamics Models

Self-Improvement Dynamics Model Learning from Human Cross-Embodiment

제출: 2026년 6월 19일 (v1) · CC BY 4.0

한 줄 요약 (TL;DR)

인간 영상에서 embodiment-불변의 행동·동역학(dynamics)·가치(value) 표현을 학습해, 이를 로봇에 접지하고 실세계 실패로부터 스스로 개선하게 만드는 프레임워크. 핵심은 DGAC(Dynamics-Guided Action Correction) — 온라인 인간 개입 없이 실패한 rollout 상태를 교정 지도신호로 변환하는 학습 불필요(training-free) 플러그인 후처리 모듈. 7개 실세계 조작 태스크, 모바일 매니퓰레이터+고정 팔 두 embodiment에서 검증. 평균 성공률 40.0% → 81.0%, 강력한 π0.5 백본에서도 SFT 62.7% → 88.0%.

1배경 · 문제의식

인간 영상은 확장 가능한 사전지식의 원천이나, 대부분 로봇 배포에만 쓰였다. 이 논문의 질문: 인간 영상 사전지식이 배포뿐 아니라 실세계 실패로부터의 자기 개선(self-improvement)까지 지원할 수 있는가? 온라인 인간 보상/개입 없이 실패를 지도신호로 바꾸는 것이 관건.

2핵심 Contribution

  • 재사용 가능한 인간 영상 사전지식: 로봇 형태를 넘나드는 embodiment-불변의 행동·동역학·가치 표현을 확장 가능한 인간 영상에서 학습, 자율적 정책 개선의 예측 기반 제공.
  • 실패→지도 학습 (DGAC): 온라인 인간 개입 없이 실패 rollout 상태를 교정 지도신호로 변환하는 학습 불필요 방법.
  • 정책 불변(policy-agnostic) 개선: DGAC가 서로 다른 정책 백본에 붙는 plug-and-play 후처리 모듈로 작동.
  • 교차 embodiment 자기개선 실증: 모바일 매니퓰레이터(Stretch 3)와 고정 팔(Franka Panda)에서 7개 실세계 조작 태스크 검증.

3Method

embodiment-불변 표현 (인간 영상에서)

  • 행동: at=[ξt, ct], ξt∈SE(3) 절대 6-DoF 손목 모션 + ct∈[0,1] 스칼라 손 닫힘(손가락 구성에서 추정, embodiment별 관절 추상화).
  • 월드 상태: ot=[zt, Pt], zt=현 프레임 DINO-v3 시각 토큰 + Pt=단기 3D point flow(TAPIP3D). "비대칭 시간 설계"로 단기 의미 맥락 + 접촉 기하 포착.
  • 가치: 희소 종료 보상의 할인 리턴 Vt=γT−trT, rT∈{1,−1}.

세 모델 공동 사전학습

정책 πθ, 동역학 fφ(상태 이력+후보 행동 청크 → 미래 상태), 가치 Vψ. 정책·동역학은 flow matching(x-prediction, clean target 직접 예측), 가치는 L2 회귀. RL은 유한 지평 contextual MDP, 청크 수준 부트스트랩 advantage Â = Σr + V(ot+H) − V(ot), 개선 지표 It=1(Â>ε)로 CFGRL식 정책 추출.

사전학습 → 자기개선

자율 로봇 적응: 동결된 인간 사전학습 정책이 VLM(GPT-4o)이 제안한 원자적 지시를 실행하며 로봇 상태·자연 실패에 노출 → 동역학·가치 모델을 대상 로봇에 접지(텔레옵 불필요). 정책 초기화: 소량 텔레옵 Dtele로 앵커링, 그리퍼 뷰 관찰 + 개선 지표 It 조건.

DGAC (Algorithm 1): 실패 상태마다 ① 진행도 정렬된 성공 경험 검색 → ② value gap/유사도 임계(δv, δs) 초과 시 교정 기각 → ③ flow matching으로 N개 교정 행동 후보 샘플 → ④ 동역학 fφ로 롤아웃, 가치 Vψ로 채점 → ⑤ a*=argmaxn V 선택. 선택된 교정으로 실패 전이를 relabel → 다음 정책 업데이트의 지도신호.

4실험 결과

헤드라인: 다양한 정책 백본에서 성공률을 평균 40.0% → 81.0%로 향상. 강력한 π0.5(대규모 로봇 데이터 사전학습) 백본에서도 SFT 62.7% → 88.0%. 동일 rollout에 단순 value-filtering(RECAP식)만 쓰면 62.7%→68.0%에 그쳐, DGAC의 명시적 실패 교정이 +20.0%p 도약.
