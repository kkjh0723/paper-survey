Vesta: A Generalist Embodied Reasoning Model

한 줄 요약 (TL;DR)

로봇의 위치추정(localization) · 공간추론 · 내비게이션 · 장기 계획(long-horizon planning)을 각각의 전문(specialist) 모델로 쌓아 올리는 대신, 단일 8B 제너럴리스트 플래너 하나로 통합할 수 있음을 보인 논문. Qwen3-VL-8B를 공간 인지 중심으로 큐레이션한 대규모 SFT 데이터로 파인튜닝하고, 간단한 이미지+텍스트 메모리 하니스(memory harness)를 붙여 장기 기억 추론을 가능하게 했다. 결과적으로 개별 SOTA 전문가 대비 평균 >20%, 카테고리별 최고 전문가들의 앙상블 대비 >10%, 실제 로봇 태스크에서 성공률 +38.3% 향상.

1배경 · 문제의식

개방형 환경(open world)의 로봇은 위치추정, 공간추론, 내비게이션, 장기 계획을 매끄럽게 통합해야 한다. 최근 주류 방식은 이를 계층적 스택(hierarchical stack)으로 분해한다 — 상위의 플래너 VLM이 고수준 지시를 내리면, 하위의 전용 액션 모델(VLA)이 이를 실행한다. 그런데 학계는 각 능력을 서로 고립된 문제(silo)로 다뤄, 내비게이션 전문가·기억 전문가·추론 전문가를 따로따로 개발해 왔다.

능력마다 전문가를 하나씩 두는 앙상블 방식의 문제는 확장성이다. 모듈화는 지연(latency)을 늘리고, 추론 스택을 복잡하게 만들며, 한 전문가의 오류가 시스템 전체로 전파되는 연쇄 실패(cascading failure)에 취약하다. 저자들의 주장: 이 능력들은 단일 제너럴리스트 플래너로 통합될 수 있고, 통합되어야 한다.

2핵심 Contribution

  • Vesta — 네 가지 능력 축(위치추정·내비게이션·공간추론·기억 기반 계획)에서 각 도메인 전문가와 대등하거나 그를 능가하는 제너럴리스트 임바디드 모델.
  • 학습 레시피 — 간단한 메모리 하니스 + 공간 편향(spatially-biased) SFT 코퍼스의 결합. 단순 SFT보다 우수한 교차 태스크 일반화를 달성.
  • 실제 로봇 검증 — 양팔(bimanual) 로봇 플랫폼에서 장기·기억 집약적 태스크의 실행 성능을 크게 개선함을 실증.
  • 결론적 메시지 — 제너럴리스트 플래너가 전문가 조합에 대한 실현 가능하고, 확장 가능하며, 더 선호할 만한 대안임을 입증.

3Method

베이스 모델 & 구조

Qwen3-VL-8B를 파인튜닝(SFT). 포인트·박스 같은 구조적 출력도 자유 답변과 동일한 언어 헤드(language head)로 텍스트 토큰으로 디코딩한다. 계층적 제어에서 상위 플래너 역할을 맡고, 하위 액터가 실제 동작을 실행한다. 비교 대상 베이스라인도 모두 동일한 8B 규모.

Localization은 "base–tail 전략"으로 구성. base는 대규모 그라운딩/검출 데이터(Objects365·COCO·LVIS)로 범용 사전지식을, tail은 에고센트릭 관찰·조작 중심 주석·시간적 상호작용 시퀀스 등 임바디드 특화 데이터를 더해 부분 관찰·시점 변화·행동 관련성에 적응시킨다. Navigation은 R2R 방식 VLN으로, 출력이 (a) 픽셀 목표점 (u,v), (b) 좌/우 회전 시퀀스, (c) 정지 중 하나. Embodied Reasoning은 "무엇을·어디에·어떻게·언제(what/where/how/when)"를 통합 인터페이스로 다룬다.

멀티모달 메모리 하니스 (핵심 아이디어)

에고센트릭 영상에서 장기 계획은 비마르코프(non-Markovian) 문제 — 다음 서브태스크 at가 전체 이력에 의존한다. 무한정 긴 컨텍스트를 피하기 위해 상태를 st = Φ(ot, Mt, g)로 근사한다. 여기서 Mt는 메모리 하니스가 압축한 컨텍스트.

각 서브태스크 예측 전에 4단계 Chain-of-Thought를 수행한다: ① Observation 무엇이 보이나 ② Progress 얼마나 진행됐나 ③ Reasoning 다음에 무엇을 왜 ④ Action 예측 서브태스크. 이 중 Action만 메모리에 기록된다.

메모리는 각 스텝을 튜플 mi = ⟨i, τi, oi, ai, g⟩(인덱스·타임스탬프·관찰·예측 서브태스크·목표)로 저장하고, 매 스텝 이를 프롬프트에 다시 주입한다. 과거 이미지는 최대 K장으로 제한(샘플링 연산자 |St|≤K)하며, 첫 프레임은 항상 유지해 초기 상태를 보존한다. 프레임 샘플링은 균등(uniform) vs 최근 편향(recency-biased) 두 방식. 즉 과거 이미지 프레임과 과거 서브태스크의 텍스트 캐시를 교차로 엮는 미니멀한 설계다.

4다루는 능력 (4개 축)

  • Localization — 그라운딩·검출, 접촉/조작 지점 pointing. 지각↔행동 인터페이스.
  • Navigation — 지시 기반 VLN(R2R 방식). 픽셀 목표점 / 회전 시퀀스 / 정지 출력.
  • Embodied Reasoning — 어포던스·배치 예측, 조작 궤적 생성, 진행도 추정, VQA.
  • Action Planning with Memory — 에고센트릭 영상 기반 장기·기억 조건부 다음-서브태스크 계획.

5실험 결과

헤드라인: 4개 능력 평균으로 최강 단일 베이스라인 대비 >20점, 카테고리별 최고 전문가 앙상블(oracle) 대비 >10점 우위. 실제 로봇 기억 집약 태스크에서 평균 성공률 +38.3%.

실제 로봇 (양팔 YAM 그리퍼, 액터 = Gr00t-N1.6)

기억·추론 집약 3개 태스크(Find Object 열었던 서랍 기억, Count Fruits 개수 추적, Memorize Candy 상자 내용물 기억), 각 20회 평가. Vesta 플래너 사용 시 액터 단독 대비 평균 성공률 +38.3%, Qwen3-VL 플래너 대비 +25%. 표본 기준 통계적으로 >4σ 유의. 실패의 다수는 플래너가 아니라 액터(동작 품질·언어 추종) 오류에서 기인.

양의 전이(positive transfer): 통합 모델이 각 전문가를 자기 도메인에서도 능가(내비 +1.4 SR, 임바디드 평균 +3.9). 전문가는 도메인 밖에서 0점.

  • Transition 샘플링: 행동 전환 구간은 전체의 ~25%로 희소. 이 구간을 2배 오버샘플링할 때 전체 정확도 최고(75.9). 3배는 이득 미미하고 실행 정확도 소폭 하락.
  • 메모리 모달리티: 이미지+텍스트 결합(균등 샘플)이 최고(75.9). 이미지-only는 진행도 판단 실패로 조기 전환, 텍스트-only는 "현 작업 계속" 예측에 편향. 균등 vs 최근편향 샘플링은 대등 → 모달리티 조합만 맞으면 프레임 선택 정책은 병목이 아님.

7한계 · 비고

  • 실제 로봇 평가는 시간 소모가 크고 액터/플래너 실패가 뒤섞여, 대안으로 오프라인 벤치마크를 설계함.
  • 로봇 시간 제약으로 학계 벤치마크용 전문가 베이스라인은 실로봇에서 평가하지 못함.
  • 내비게이션에서 SPL·NE는 여전히 전문가 InternVLA-N1에 근소 열세(전체적으로는 대등).
