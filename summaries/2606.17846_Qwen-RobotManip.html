Qwen-RobotManip: Alignment Unlocks Scale for Robotic Manipulation Foundation Models

Qwen Team (Alibaba) · 기술 리포트

arXiv abstract PDF 프로젝트/블로그 GitHub arXiv:2606.17846
VLA Foundation Model Cross-Embodiment Alignment OOD Generalization

제출: 2026년 6월 17일 (v2) · 백본 Qwen3.5-4B VLM + DiT action expert

한 줄 요약 (TL;DR)

"정렬(alignment)이 스케일을 푼다." 이질적 embodiment를 통합하는 정렬 프레임워크(정규 80차원 상태-행동 표현 + 카메라 프레임 델타 pose + 실행 이력 기반 in-context 적응)와 ~38,100시간의 순수 오픈소스 데이터(사유 데이터 무수집, 인간→로봇 합성 24,808h 포함)로 학습한 조작 파운데이션 모델. 표준 in-domain 벤치는 사전학습 효과를 못 가린다고 주장, OOD 평가를 north star로 제시. LIBERO-Plus 91.4%(π0.5 84.4), zero-shot 교차 embodiment 평균 23.9%(π0.5의 3.2배), RoboChallenge Table30-v1 제너럴리스트 트랙 1위.

1배경 · 문제의식

로봇 조작 파운데이션 모델의 스케일업을 막는 근본 문제는 embodiment 간 표현 이질성(representational heterogeneity). 서로 다른 로봇의 상태·행동·카메라·형태가 제각각이라 대규모 다출처 데이터를 하나의 정책으로 녹이기 어렵다. 저자들의 처방은 세 층위의 정렬이다.

2핵심 Contribution

  • 교차 embodiment 통합 정렬 프레임워크: ① 차원별 이진 마스킹을 쓰는 정규 80차원 상태-행동 표현, ② EEF 행동을 시각 도메인에 접지하는 카메라 프레임 델타 pose, ③ 에피소드 내 실행 이력을 암묵적 embodiment 식별자로 쓰는 in-context 적응. + 조작·시각언어를 공동 최적화하는 이중 스트림 co-training.
  • 확장형 다출처 코퍼스(~38,100h): 전부 오픈소스 로봇 데이터 + 에고센트릭 인간 영상, 사유 데이터 무수집. 어떤 에고센트릭 데모든 15개 로봇 플랫폼 궤적으로 변환하는 인간→로봇 합성 파이프라인(~24,808h).
  • VLA 일반화 평가의 새 기준: in-domain 벤치는 사전학습 품질을 못 가린다고 주장. OOD 세팅(LIBERO-Plus, RoboTwin-Clean2Rand, RoboCasa365, EBench) + 신규 벤치 RoboTwin-IF(지시 추종)·RoboTwin-XE(zero-shot 교차 embodiment).
  • 실로봇 검증: 4개 물리 플랫폼(AgileX ALOHA, Franka, UR, ARX)에서 in-domain/OOD/few-shot/zero-shot 검증. RoboChallenge Table30-v1 제너럴리스트 트랙 1위(상대 +20%).

3Method

구조 (분리형)

Qwen-VL 백본(Qwen3.5-4B, Dvlm=2560) + flow-matching Diffusion Transformer(DiT) action expert(N=10 블록, Dact=768, 12 heads). 각 DiT 블록: 상태+행동 토큰 self-attention → VLM hidden state로 cross-attention(짝수 블록=시각, 홀수 블록=언어 교차) → SwiGLU FFN. flow-matching(t~Beta(1,1.5)), 추론은 4스텝 Euler로 저지연 제어.

교차 embodiment 상태/행동 표현

80차원 정규 벡터 = 팔당 29차원 블록 2개 + 예약 22차원. 팔 블록: 관절위치(7) + EEF pose(9=위치3+6D 회전) + 그리퍼(1) + 손 관절(12). 상태·관절행동은 절대값, EEF 행동은 상대 델타. embodiment마다 다른 부분집합만 채우고, zero-pad 차원은 차원별 이진 마스크로 손실에서 제외.

통합 EEF 모션 예측: 카메라 프레임 델타 pose로 시각적으로 유사한 동작을 수치적으로 가깝게 만들고, Camera Positional Encoding(CaPE)로 카메라 기하를 DiT cross-attention에 주입. EEF 타입 임베딩(단팔/양팔 좌우/에고센트릭 헤드/모바일 베이스) + 캘리브레이션 가용 플래그를 adaptive layernorm으로 조건화.

in-context 정책 적응

최근 실행 컨텍스트 청크 윈도우(관찰, 상태, K-step 행동 삼중항)를 조건으로. 이력 프레임은 VLM 비전 인코더에 prepend. 확률적 컨텍스트 샘플링(최근이 아닌 무작위 에피소드 위치에서 윈도우 추출)으로 행동 복사 shortcut 붕괴 방지.
